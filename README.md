# Text-Data-Analysis

In this repository, the code is presented as it was used for the development and execution of the project. It will be further reduced to the essential minimum to improve final readability. However, the functions and additional calls do not affect the overall process flow and still produce the desired results.

At this point, two important aspects need to be mentioned:

1. The data is no longer available. It was temporarily generated and stored in a secure company environment. At the end of the project, it was properly deleted to comply with data protection regulations. For this reason, these exact results can no longer be reproduced.

2. “GeVo,” which is used throughout the code and was not translated from German, stands for “Geschäftsvorfall” (business transaction). It refers to a process in the system that modifies contracts in a desired way, for example due to a job change.
